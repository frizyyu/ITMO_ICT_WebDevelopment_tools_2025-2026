Лабораторная работа 3¶

Отчет подготовлен для публикации через GitHub Pages.

Материалы¶

  • Открыть отчет в браузере
  • Скачать отчет в DOCX

Как собрать сайт локально¶

Из папки Lr3:

pip install mkdocs
mkdocs build

После сборки появится папка site/ со статическими HTML, CSS и JS-файлами.

Для локального просмотра:

mkdocs serve

Затем открыть:

http://127.0.0.1:8000

Как опубликовать на GitHub Pages¶

В проект добавлен workflow:

.github/workflows/pages.yml

Он устанавливает MkDocs, запускает mkdocs build --strict в папке students/K3340/Vasilev_Artem/Lr3 и публикует папку site/ в GitHub Pages.

Важно: GitHub Actions видит workflow только из папки .github/workflows в корне репозитория. Если папка Lr3 не является корнем GitHub-репозитория, нужно перенести файл Lr3/.github/workflows/pages.yml в .github/workflows/pages.yml корня репозитория или использовать локальную сборку mkdocs build и публиковать содержимое папки site/.

DOCX-файл публикуется как обычный статический файл для скачивания. Для просмотра отчета прямо в браузере используется Markdown-версия.
